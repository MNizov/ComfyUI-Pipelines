# ControlNet Depth (карта глубины) 📐✨

Пайплайн для генерации изображений с использованием ControlNet на основе карты глубины (Depth). Позволяет сохранять композицию и пространственную структуру референсного изображения, изменяя при этом стиль, цвета и детали. 🖼️➡️📏➡️✨

## Описание работы пайплайна 🔄

Пайплайн использует Depth Anything V2 для создания карты глубины и ControlNet для контроля структуры изображения:

### Этап 1: Подготовка референса и карты глубины 📏
1. **Загрузка референсного изображения** 📥 — `LoadImage` загружает изображение, структуру которого нужно сохранить
2. **Создание карты глубины** 📐 — `DepthAnythingV2Preprocessor` анализирует изображение и создает карту глубины (черно-белое изображение, где белое = близко, черное = далеко)
   - Модель: `depth_anything_v2_vitl.pth`
   - Разрешение: `384px` (для предобработки)

### Этап 2: Подготовка модели и ControlNet 🎛️
3. **Загрузка базовой модели** 📦 — `CheckpointLoaderSimple` загружает чекпоинт (например, `RealVisXL_V5.0_Lightning_fp16.safetensors`)
4. **Загрузка ControlNet** 🔧 — `ControlNetLoader` загружает модель ControlNet для карт глубины (`diffusion_pytorch_model.safetensors`)
5. **VAEEncode** 🔄 — кодирует карту глубины в латентное пространство (используется как исходное изображение для img2img)

### Этап 3: Применение ControlNet и генерация ✨
6. **CLIPTextEncode (позитивный)** ✅ — промпт для желаемого содержимого
7. **CLIPTextEncode (негативный)** ❌ — промпт для исключения артефактов
8. **ControlNetApplyAdvanced** 🔗 — применяет ControlNet к CONDITIONING, используя карту глубины для контроля структуры
   - `strength`: `0.75` — сила влияния ControlNet
   - `start_percent`: `0.183` — начало применения
   - `end_percent`: `0.853` — конец применения
9. **KSampler** 🎲 — выполняет генерацию с режимом img2img (`denoise = 0.82`), сохраняя композицию референса
10. **VAEDecode** 🔓 — декодирует латент в итоговое изображение
11. **SaveImage** 💾 — сохраняет результат с префиксом `controlnet_output`

## Используемые ноды (переменные для настройки) 🎛️

### Загрузка исходных данных 📂

#### `LoadImage` 🖼️
Загружает референсное изображение для сохранения структуры.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `image` | Путь к изображению | `input.png` |

#### `CheckpointLoaderSimple` 📦
Загрузчик базовой модели.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `ckpt_name` | Название файла чекпоинта | `RealVisXL_V5.0_Lightning_fp16.safetensors` |

### ControlNet и предобработка 🎛️

#### `DepthAnythingV2Preprocessor` 📐
Создает карту глубины из референсного изображения.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `ckpt_name` | Модель Depth Anything V2 | `depth_anything_v2_vitl.pth` |
| `resolution` | Разрешение предобработки | `384` |

#### `ControlNetLoader` 🔧
Загружает ControlNet модель.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `control_net_name` | Название ControlNet | `diffusion_pytorch_model.safetensors` |

#### `ControlNetApplyAdvanced` 🔗
Применяет ControlNet к CONDITIONING.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `strength` | Сила влияния ControlNet | `0.75` |
| `start_percent` | Начало применения | `0.183` |
| `end_percent` | Конец применения | `0.853` |

### Промпты 📝

#### `CLIPTextEncode` (позитивный) ✅
Что должно быть на изображении.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `text` | Позитивный промпт | `portrait of a woman, detailed face, soft studio lighting, photorealistic, elegant red hat, cinematic composition, depth of field, 8k, highly detailed, bright, well-lit, soft lighting, bright studio lighting, high key lighting` |

#### `CLIPTextEncode` (негативный) ❌
Чего не должно быть на изображении.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `text` | Негативный промпт | `blurry, low quality, deformed, bad anatomy, extra limbs, fused body, bad hands, missing fingers, distorted face, ugly face, bad proportions, cropped, worst quality, jpeg artifacts, dark, shadow, low light, underexposed, dim` |

### Параметры генерации ⚙️

#### `KSampler` 🎲
Основной сэмплер (режим img2img).

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `seed` | Сид генерации | `897248593402398` или `randomize` |
| `steps` | Количество шагов | `10` |
| `cfg` | Масштаб классификатора | `2.5` |
| `sampler_name` | Алгоритм сэмплирования | `dpmpp_2m` |
| `scheduler` | Расписание шума | `karras` |
| `denoise` | Сила денойзинга __(важно! <1 для сохранения структуры)__ | `0.82` |

### Сохранение результатов 💾

#### `SaveImage` 💿
Сохраняет итоговое изображение.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `filename_prefix` | Префикс имени файла | `controlnet_output` |

## Быстрый старт 🚀

1. Загрузите пайплайн в ComfyUI (файл `workflow.json`) 📂
2. Убедитесь что у вас установлены необходимые ноды: **ComfyUI ControlNet Aux** 📦
3. В `LoadImage` укажите путь к вашему референсному изображению (структура которого нужна) 🖼️
4. В `CheckpointLoaderSimple` укажите путь к вашему чекпоинту ⚠️
5. Убедитесь что у вас есть модели:
   - Depth Anything V2 (`depth_anything_v2_vitl.pth`) в папке `ComfyUI/models/controlnet/` 📁
   - ControlNet для глубины (`diffusion_pytorch_model.safetensors`) в папке `ComfyUI/models/controlnet/` 📁
6. Отредактируйте промпты под желаемый результат ✏️
7. Настройте ключевые параметры:
   - `strength` в ControlNetApplyAdvanced — сила контроля структуры (0.6-0.8) 🎛️
   - `denoise` в KSampler — степень изменения (0.7-0.9) ⚙️
8. Нажмите "Queue Prompt" для генерации ▶️
9. Результат появится в папке `ComfyUI/output` с префиксом `controlnet_output` ✨

## Настройка баланса "структура vs стиль" ⚖️

### ControlNet `strength`
| Значение | Эффект |
|----------|--------|
| `0.3-0.5` | Слабое сохранение структуры, больше свободы |
| `0.6-0.8` | Хорошее сохранение композиции __(рекомендуется)__ |
| `0.9-1.0` | Сильное сохранение, почти копия структуры |

### KSampler `denoise`
| Значение | Эффект |
|----------|--------|
| `0.5-0.7` | Небольшие изменения, близко к референсу |
| `0.8-0.9` | Значительные изменения, новый стиль __(рекомендуется)__ |
| `1.0` | Полная перегенерация (теряется связь с референсом) |

### Комбинации
| Цель | ControlNet `strength` | `denoise` |
|------|----------------------|-----------|
| Сохранить позу, изменить стиль | `0.7-0.8` | `0.8-0.9` |
| Легкая стилизация | `0.5-0.6` | `0.6-0.7` |
| Точное сохранение композиции | `0.8-0.9` | `0.5-0.6` |

## Важные замечания ⚠️

- Для работы пайплайна необходима кастомная нода: **ComfyUI ControlNet Aux** 📦
- Depth Anything V2 модели скачиваются автоматически или вручную в папку `ComfyUI/models/controlnet/` 🌐
- ControlNet для глубины должна быть совместима с вашей базовой моделью (SD1.5/SDXL) 🔄
- Карта глубины создается в реальном времени — можно использовать любые изображения 🖼️
- Разрешение в `DepthAnythingV2Preprocessor` влияет на качество карты глубины и скорость обработки ⚡
- Пайплайн работает в режиме **img2img** — латент создается из карты глубины, а не из пустоты 📐
