# IMG2IMG + Inpainting (с маской) 🎨🔧✨

Продвинутый пайплайн для редактирования изображений с автоматическим выделением объектов (детекшн + SAM) и инпейнтингом (дорисовкой) по маске. Поддерживает ControlNet для лучшего сохранения структуры. 🖼️➡️✏️

## Описание работы пайплайна 🔄

Пайплайн автоматически обнаруживает объекты на изображении, создает маску и выполняет инпейнтинг (замену/дорисовку) выделенной области:

### Этап 1: Детекция и создание маски 🎯
1. **Загрузка изображения** 📥 — `LoadImage` загружает исходное изображение для редактирования
2. **Детекция объектов** 🔍 — `UltralyticsDetectorProvider` загружает YOLO-модель для обнаружения объектов (по умолчанию `person_yolov8m-seg.pt`)
3. **BboxDetectorSEGS** 📦 — создает bounding box'ы для обнаруженных объектов
4. **SAM (Segment Anything)** ✂️ — `SAMLoader` загружает модель SAM (`sam_vit_b_01ec64.pth`)
5. **SAMDetectorSegmented** 🧩 — создает точную маску объекта на основе bounding box'ов
6. **FeatherMask** 🌫️ — размывает края маски для плавного перехода (значения `14, 8, 14, 14`)
7. **ThresholdMask** ⚖️ — применяет пороговое значение к маске (`0.1`)
8. **MaskToImage** 🖼️ — преобразует маску в изображение для предпросмотра
9. **PreviewImage (маска)** 👁️ — показывает созданную маску

### Этап 2: Подготовка к инпейнтингу 🛠️
10. **InpaintPreprocessor** 🎭 — создает изображение с замаскированной областью (черная область для инпейнтинга)
11. **ControlNetLoader** 🎛️ — загружает ControlNet модель (`diffusion_pytorch_model.safetensors`)
12. **ControlNetApplyAdvanced** 🔗 — применяет ControlNet к CONDITIONING для сохранения структуры
13. **VAEEncodeForInpaint** 🔄 — кодирует изображение в латентное пространство с учетом маски (параметр `grow_mask_by` = `3`)
14. **InvertMask** 🔄 — инвертирует маску (область для генерации становится белой)

### Этап 3: Генерация (инпейнтинг) ✨
15. **CLIPTextEncode (позитивный)** ✅ — промпт для желаемого содержимого в замаскированной области
16. **CLIPTextEncode (негативный)** ❌ — промпт для нежелательных элементов
17. **KSampler** 🎲 — выполняет генерацию только в области маски
18. **VAEDecode** 🔓 — декодирует латент в итоговое изображение
19. **SaveImage** 💾 — сохраняет результат с префиксом `inpaint_result`

## Используемые ноды (переменные для настройки) 🎛️

### Загрузка исходных данных 📂

#### `LoadImage` 🖼️
Загружает изображение для редактирования.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `image` | Путь к изображению | `photo.png` |

#### `CheckpointLoaderSimple` 📦
Загрузчик базовой модели для генерации.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `ckpt_name` | Название файла чекпоинта | `RealVisXL_V5.0_Lightning_fp16.safetensors` |

### Детекция объектов и создание маски 🎯

#### `UltralyticsDetectorProvider` 🔍
Загружает модель детекции объектов.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `model_name` | Название YOLO-модели | `segm/person_yolov8m-seg.pt` |

#### `BboxDetectorSEGS` 📦
Настраивает параметры детекции.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `threshold` | Порог уверенности детекции | `0.24` |
| `dilation` | Расширение bounding box | `11` |
| `crop_factor` | Фактор обрезки | `3` |
| `drop_size` | Минимальный размер объекта | `10` |
| `labels` | Фильтр по классам | `all` |

#### `SAMLoader` ✂️
Загружает модель SAM для сегментации.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `model_name` | Название SAM-модели | `sam_vit_b_01ec64.pth` |
| `device_mode` | Режим устройства | `Prefer GPU` |

#### `SAMDetectorSegmented` 🧩
Создает маску объекта.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `detection_hint` | Режим детекции | `mask-point-bbox` |
| `dilation` | Расширение маски | `2` |
| `threshold` | Порог уверенности | `0.3` |
| `bbox_expansion` | Расширение bounding box | `0` |
| `mask_num` | Количество масок | `0.41` |
| `post_dilation` | Пост-расширение | `False` |

#### `FeatherMask` 🌫️
Размывает края маски для плавного перехода.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `left` | Размытие слева | `14` |
| `top` | Размытие сверху | `8` |
| `right` | Размытие справа | `14` |
| `bottom` | Размытие снизу | `14` |

#### `ThresholdMask` ⚖️
Применяет порог к маске.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `threshold` | Пороговое значение | `0.1` |

### ControlNet 🎛️

| Поле | Описание | Пример значения |

| `control_net_name` | Название ControlNet | `diffusion_pytorch_model.safetensors` |

| Поле | Описание | Пример значения |
|------|----------|-----------------|

### Промпты 📝

#### `CLIPTextEncode` (позитивный) ✅
Что должно появиться в замаскированной области.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `text` | Позитивный промпт | `beautiful sunset over the ocean, tropical beach, palm trees, golden hour, 8k, photorealistic` |

#### `CLIPTextEncode` (негативный) ❌
Чего не должно быть в замаскированной области.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `text` | Негативный промпт | `blurry, low quality, deformed, bad anatomy, extra limbs, fused body, bad hands, missing fingers, distorted face, ugly face, bad proportions, cropped` |

### Инпейнтинг 🎨

#### `VAEEncodeForInpaint` 🔄
Кодирует изображение с учетом маски.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `grow_mask_by` | Расширение маски (пиксели) | `3` |

#### `KSampler` 🎲
Основной сэмплер для генерации.

| Поле | Описание | Пример значения |
|------|----------|-----------------|
| `seed` | Сид генерации | `339700731900317` или `randomize` |
| `steps` | Количество шагов | `9` |

### Сохранение результатов 💾

## Быстрый старт 🚀

1. Загрузите пайплайн в ComfyUI (файл `IMG2IMG + Inpainting (с маской) (7).json`) 📂
2. В ноде `LoadImage` укажите путь к вашему изображению для редактирования 🖼️
3. В `CheckpointLoaderSimple` укажите путь к вашему чекпоинту ⚠️
4. Убедитесь что у вас установлены необходимые модели:
   - YOLO для детекции (`person_yolov8m-seg.pt`) 👤
   - SAM для сегментации (`sam_vit_b_01ec64.pth`) ✂️
   - ControlNet для инпейнтинга (`diffusion_pytorch_model.safetensors`) 🎛️
5. При необходимости отредактируйте промпты для желаемого содержимого ✏️
6. Настройте параметры детекции в `BboxDetectorSEGS` (если нужно выделить другой объект) 🎯
7. Нажмите "Queue Prompt" для генерации ▶️
8. Проверьте маску в окнах предпросмотра 👀
9. Результат появится в папке `ComfyUI/output` с префиксом `inpaint_result` ✨

## Важные замечания ⚠️

- Для работы пайплайна необходимы дополнительные ноды: **ComfyUI Impact Pack** и **ComfyUI ControlNet Aux** 📦
- YOLO модели для детекции скачиваются автоматически при первом использовании 🌐
- SAM модель необходимо скачать вручную и поместить в папку `ComfyUI/models/sams/` 📁
- ControlNet для инпейнтинга должна быть совместима с вашей базовой моделью 🔄
- Значение `denoise=1` означает полную перерисовку области маски. Для сохранения фона можно уменьшить до `0.7-0.9` ⚙️
- FeatherMask настраивает плавность перехода: большие значения = более плавные края 🌫️
